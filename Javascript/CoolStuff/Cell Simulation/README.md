# Cell Simulation 
Cell Simulation with P5.js, based on the principles of "The Game of Life" written by Jonh Conway.

## Demo
Copy and past the code to https://editor.p5js.org/, and click play.

## Options
#### Add walls
Change the variable "asWalls" to true
